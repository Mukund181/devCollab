## A collaborative platform for developers - backend project

Covers : 
1) Mongodb crud and relationships 
2) Authentication and authorization with jwt
3) Express middlewares like logging , validation, cookie-parsing and error handling
4) File upload with multer and mailing with nodemailer- sending verification email for new user registraton and reset password link for existing users 
5) Real-time communication with socket.io
6) Redis - caching and rate limiting
7) Docker - containerization
8) Kubernetes - orchestration
9) MongoDB Atlas - cloud database
10) AWS deployment with ec2 and load balancing
